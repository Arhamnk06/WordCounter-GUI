# WordCounter-GUI
This project is a Word Counter application built using Java Swing and 2D graphics.
Users can enter text into a text area, click a button to analyze it, and view the results both numerically and visually through an interactive bar chart.

The program demonstrates GUI development, custom drawing, and event-driven programming in Java.

✨ Features

Text Analysis
	•	Counts:
	•	Total words
	•	Total characters
	•	Total lines
	•	Displays results through JLabels
	•	Updates instantly when the user clicks Count

Interactive 2D Bar Chart
	•	Custom JPanel (StatsDisplayPanel) displays:
	•	Word count bar
	•	Character count bar
	•	Line count bar
	•	Bars scale proportionally based on values
	•	Bars highlight/change color on mouse hover

User Controls
	•	Count button → analyzes text and updates graphics
	•	Clear button → resets text and stats
	•	Hovering over bars → changes bar color
